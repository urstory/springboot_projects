ViewResolver 예제

🎯 ViewResolver 작동 과정

ViewResolver를 통해 HTML 페이지가 반환됩니다.

📋 처리 흐름

  1. Controller 반환: "home" (View 이름)
  2. DispatcherServlet: View 이름을 ViewResolver에 전달
  3. ThymeleafViewResolver: "home" → "/templates/home.html" 변환
  4. 최종 응답: HTML 페이지 반환

🔗 예제 페이지

사용자 목록 (ViewResolver) 사용자 상세 (ViewResolver) API 사용자 목록 (MessageConverter) API 정보 (MessageConverter)

💡 학습 포인트

  • ViewResolver: View 이름 → HTML 페이지 변환
  • MessageConverter: 객체 → JSON 응답 변환
  • @Controller: View 이름 반환 (HTML 응답)
  • @RestController: 객체 반환 (JSON 응답)
