ViewResolver vs MessageConverter

  • 현재 페이지 (ViewResolver): HTML 테이블로 사용자 목록 표시
  • API 버전 (MessageConverter): JSON 형태로 동일한 데이터 반환

같은 데이터를 다른 방식으로 처리하는 것을 확인할 수 있습니다!
